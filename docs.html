Documentation

MONAI offers serveral frameworks, and we are adding to them all the time. Here, you’ll find the information you need to decide on what framework works best for your needs.

MONAI Core

Domain-optimized foundational capabilities for developing healthcare imaging training workflows in a native PyTorch paradigm.

MONAI Label

Image Labeling and Learning Tool that helps create annotated datasets and build AI models in a standardized MONAI paradigm.

MONAI Deploy

A framework and associated tools to design, verify and analyze the performance of AI-driven applications in the healthcare domain.
